# Hello world lesson

1. Install rust:

    ```
    curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh

    rustup --version
    ```

2. Write "hello world" program

    See `main.rs`

3. Compile the program

    ```
    rustc main.rs
    ```

4. Run the program

    ```
    ./main
    ```
